This software is designed for **DMA**.

| Component | Requirement |
| :--- | :--- |
| 🎮 **Game PC** | The primary computer where the game is running. |
| 💻 **Second PC** | A secondary laptop or PC to run the radar/software. |
| 🔌 **DMA Card** | A compatible Hardware DMA Card installed in the Game PC. |
